# Assessment Chapters

This section contains assessments and projects to evaluate your understanding of the course material.

## Available Assessments

- [ROS 2 Project](./ros2-project.md)

## Assessment Types

Throughout this course, you'll encounter various types of assessments:

- **Hands-on Projects**: Build real robotics applications
- **Code Reviews**: Analyze and improve robot control code
- **Simulations**: Test your robots in virtual environments
- **Capstone Project**: Integrate everything you've learned

Each assessment is designed to reinforce your learning and provide practical experience with humanoid robotics.
