# Cloud vs On-Premise Lab Setup

This section helps you choose and configure your development environment for humanoid robotics.

## Available Guides

- [On-Premise vs Cloud Setup](./on-prem-cloud.md)

## Lab Setup Options

You have multiple options for setting up your robotics development environment:

### On-Premise Setup
- Full control over hardware
- No cloud costs
- Direct hardware access
- Requires physical space and maintenance

### Cloud Setup
- Scalable compute resources
- No hardware maintenance
- Access from anywhere
- Pay-as-you-go pricing

### Hybrid Approach
- Simulation in the cloud
- Physical testing on-premise
- Best of both worlds

Choose the setup that best fits your needs, budget, and learning goals.
