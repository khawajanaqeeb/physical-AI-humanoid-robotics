# AI & Machine Learning Expert Skill

## Purpose
Apply deep learning, reinforcement learning, and computer vision expertise to create content about AI-driven robotic systems.

## Expertise Areas
- **Deep Learning Architectures**: CNNs, RNNs, Transformers, Vision Transformers
- **Reinforcement Learning**: Policy gradient methods, Q-learning, actor-critic, PPO, SAC
- **Robot Perception**: Object detection, segmentation, tracking, 3D vision
- **SLAM & Localization**: Visual SLAM, LiDAR SLAM, sensor fusion, mapping
- **Motion Planning**: Path planning, trajectory optimization, collision avoidance
- **Sim-to-Real Transfer**: Domain randomization, domain adaptation, reality gap
- **Dataset Engineering**: Data collection, annotation, augmentation, synthetic data

## When to Use This Skill
- Explaining neural network architectures for robot perception
- Describing reinforcement learning for robotic control
- Creating content about vision systems and sensor processing
- Developing chapters on learning-based control and planning
- Explaining sim-to-real transfer techniques

## Output Guidelines
- Explain AI/ML concepts in the context of physical robotics
- Provide clear mathematical notation for algorithms (loss functions, updates)
- Include pseudocode and implementation considerations
- Discuss training strategies, hyperparameters, and practical challenges
- Compare different approaches with pros/cons
- Reference state-of-the-art research and papers
- Address computational requirements and real-time constraints

## Content Structure
1. **Algorithm Overview**: What the method does and why it's useful
2. **Mathematical Foundation**: Equations, optimization objectives, theory
3. **Implementation Details**: Network architectures, training procedures
4. **Robot Application**: How it's applied in humanoid/physical AI systems
5. **Challenges & Solutions**: Common issues and mitigation strategies
6. **Code Examples**: Python/PyTorch snippets, ROS integration
